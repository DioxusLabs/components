Build with DioxusUI

DioxusUI is a collection of styled, accessible Dioxus components designed to be copied into your app. Use the CLI when you want the fastest path, or copy the source when you want complete ownership.

How it works

Copy-first components

Each component ships as source code and CSS you can keep, edit, and theme inside your own project.

Shared theme tokens

Import the theme CSS once, then component styles use the same color, focus, and state variables.

Dioxus primitives underneath

The styled components are built on reusable primitives for accessibility, keyboard interaction, and state.

Add a component

Run the add command from your Dioxus app. Swap the final name for any component in the sidebar.

dx components add button

If you do not have the Dioxus CLI yet, install it once with cargo install dioxus-cli.

Recommended workflow

  1. Pick a component from the sidebar or catalog.
  2. Preview the default example and variants.
  3. Run the CLI command shown on the component page.
  4. Customize the generated Rust and CSS to fit your app.
